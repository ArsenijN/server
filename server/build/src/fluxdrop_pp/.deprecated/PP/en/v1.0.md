**FluxDrop Privacy Policy – Version 1.0**

**Last Updated: July 8, 2025**

**1. Introduction**

This Privacy Policy describes how FluxDrop, operated by ArsenijN, collects, uses, stores, and protects your personal information when you use our file sharing service. By using FluxDrop, you agree to the collection and use of information in accordance with this policy.

**2. Information We Collect**

FluxDrop collects the following types of information to provide and improve our service:

**2.1 Account Information:**
*   Email address (required for account registration)
*   Account credentials (username, password hashes)
*   Account type and subscription details
*   User preferences and settings

**2.2 Usage Data:**
*   IP addresses and geographical location data
*   Date and time of login and service access
*   Device information (browser type, operating system, device name)
*   File upload and download activity
*   Service usage patterns and performance metrics
*   Error logs and diagnostic information

**2.3 File Data:**
*   Uploaded files and their metadata
*   File sharing settings and access controls
*   Download history and link usage statistics

**3. How We Use Your Information**

FluxDrop uses collected information for the following purposes:

**3.1 Service Provision:**
*   Account creation and management
*   File storage and sharing functionality
*   User authentication and security
*   Customer support and communication

**3.2 Service Improvement:**
*   Performance monitoring and optimization
*   Issue investigation and troubleshooting
*   Feature development and enhancement
*   Usage analytics and service statistics

**3.3 Security and Compliance:**
*   Rate limiting and abuse prevention
*   Account security monitoring
*   Legal compliance and regulatory requirements
*   Fraud prevention and detection

**4. Data Storage and Security**

**4.1 Encryption:**
*   All user data and credentials are encrypted using AES-256 encryption
*   File contents are encrypted when Password Access feature is enabled
*   Encryption keys are managed through a secure "head key" system

**4.2 Access Control:**
*   The "head key" (master AES-256 key) is stored physically and accessible only to the head of FluxDrop
*   User data cannot be decrypted without physical access to the head key
*   Access to user data is strictly controlled and logged

**4.3 Data Centers:**
*   Data is stored across separate data centers for redundancy
*   Physical security measures protect server infrastructure
*   Regular security audits and updates are performed

**5. Data Sharing and Disclosure**

**5.1 Limited Data Sharing:**
*   FluxDrop does not sell, trade, or share user data with third-party companies for commercial purposes
*   User data is never used for advertising or marketing by external parties
*   No personal information is shared with data brokers or advertising networks

**5.2 Governmental Requests:**
*   User data may be disclosed to governmental structures (Ukrainian or international) upon lawful request
*   Such disclosures are made only when legally required and properly documented
*   Users will be notified of data requests when legally permissible

**5.3 Legal Compliance:**
*   Data may be disclosed to comply with applicable laws and regulations
*   Information may be shared to protect FluxDrop's legal rights and interests
*   Emergency disclosures may be made to prevent immediate harm

**8. Data Retention**

**8.1 Active Data:**
*   Account information is retained while your account remains active
*   File data is retained according to your account type and storage settings
*   Usage logs are maintained for security and compliance purposes

**8.2 Deleted Data:**
*   Deleted files are retained in trash bin for standard retention period
*   Expired trash bin files are stored for minimum 14 days for transparency
*   Complete data deletion occurs during routine server maintenance cycles

**8.3 Account Deletion:**
*   Upon account termination, user data is marked for deletion
*   Complete data removal may take up to 30 days
*   Some data may be retained for legal compliance purposes
